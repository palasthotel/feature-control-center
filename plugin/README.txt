== Description ==

Add custom boolean feature meta values to posts.

== Installation ==

1. Upload `feature-control-center.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `feature-control-center` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add feature with action hook to contorl center
1. Switch feature flag in gutenberg control panel
